# Amazon Clone

A responsive Amazon homepage clone built using **HTML5** and **CSS3**. This project recreates the look and feel of Amazon's landing page to practice front-end web development skills.

## 🚀 Live Demo
https://siddharthasangani18.github.io/amazon-clone-html-css/

## ✨ Features

- Amazon-style navigation bar
- Search bar with category selector
- Hero banner section
- Product cards
- Multi-column footer
- Responsive layout
- Font Awesome icons

## 🛠️ Technologies Used

- CSS3
- Font Awesome
